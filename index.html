README

      

        This demonstrates RDF-only uses of signatures.
        Provide a document to sign, a proof structure, and a private key to produce a signed document.
        Verify the signed document against a public key below.
      

      

        Both the sign and verify button actions start with a call to a debugger, so you can follow the process step by step in DevTools.
      

      

        PRs, issues and advice welcome.
